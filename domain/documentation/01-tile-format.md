# Tile format

## T shaped

```
┘.└      │.└
...      │..
───      │.┌
 0        90
---      ---

───      ┘.│
...      ..│
┐.┌      ┐.│
180      270
```

## I shaped

```
───      │.│
...      │.│
───      │.│
 0        90
---      ---

───      │.│
...      │.│
───      │.│
180      270
```

## V shaped

```
──┐      ┘.│
..│      ..│
┐.│      ──┘
 0        90
---      ---

│.└      ┌──
│..      │..
└──      │.┌
180      270
```
